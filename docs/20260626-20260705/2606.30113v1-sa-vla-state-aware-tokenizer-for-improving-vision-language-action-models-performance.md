## 摘要
离散动作分词为自回归VLA策略提供了紧凑的接口，但从离散编码中准确恢复连续机器人动作仍具挑战。现有分词器通常将每个离散编码映射到固定的连续动作原型，忽略了机器人当前的自身状态。这一局限在操作任务中尤为显著，因为相同的动作标记在不同关节配置、物体姿态和接触条件下可能需要不同的连续控制。因此，我们提出SA-VLA，一种基于机器人状态进行动作解码的条件化状态感知动作分词器。我们研究了两种基于VQ的动作分词的状态注入机制：状态与动作特征之间的交叉注意力，以及一种轻量级状态适配器，它预测动作方向的调制因子以实现状态条件化的动作调制与重建。适配器公式通过允许每个离散标记表示一系列状态依赖的连续动作，扩展了有限码本的有效支持范围，同时保留了离散动作建模的效率和兼容性。在集成到基于LLM的VLA策略后，SA-VLA支持自回归和并行动作标记解码，且对模型接口的改动最小。在12个RoboTwin操作任务上，SA-VLA将平均成功率从0.29提升至0.56，优于最强的分词器基线。在三个真实世界任务的零样本模拟到现实实验中，它进一步将平均成功率从0.15提升至0.33，超过最强基线。这些结果表明，状态条件化的动作解码是一种简单有效的机制，用于缩小离散VLA策略中的压缩差距。

## Abstract
Discrete action tokenization provides a compact interface for autoregressive VLA policies, but accurately recovering continuous robot actions from discrete codes remains challenging. Existing tokenizers typically map each discrete code to a fixed continuous action prototype, ignoring the robot's current proprioceptive state. This limitation is particularly pronounced in manipulation, where the same action token may require different continuous controls under different joint configurations, object poses, and contact conditions. We therefore propose SA-VLA, a state-aware action tokenizer that conditions action decoding on robot state. We study two state-injection mechanisms for VQ-based action tokenization: cross-attention between state and action features, and a lightweight state adapter that predicts action-wise modulation factors for state-conditioned action modulation and reconstruction. The adapter formulation expands the effective support of a finite codebook by allowing each discrete token to represent a family of state-dependent continuous actions, while preserving the efficiency and compatibility of discrete action modeling. Integrated into an LLM-based VLA policy, SA-VLA supports both autoregressive and parallel action-token decoding with minimal changes to the model interface. On 12 RoboTwin manipulation tasks, SA-VLA improves the average success rate from 0.29 to 0.56 over the strongest tokenizer baseline. In zero-shot sim-to-real experiments on three real-world tasks, it further improves average success from 0.15 to 0.33 over the strongest tokenizer baseline. These results demonstrate that state-conditioned action decoding is a simple and effective mechanism for reducing the compression gap in discrete VLA policies.

## 1. 论文简介和核心思想
**背景**：基于自回归Transformer的视觉-语言-动作（VLA）模型通常先将连续动作离散化为“动作令牌”，再用大型语言模型（LLM）等架构生成这些令牌，最后通过一个“动作分词器”解码为连续动作。然而，现有分词器解码时往往忽略机器人的当前状态（如关节角度、物体位姿），导致解码精度受限。
**核心思想**：论文提出SA-VLA，即“**状态感知动作分词器**”。其核心是让动作解码过程**以机器人当前状态为条件**。通过一个轻量级的“状态适配器”或交叉注意力机制，为每个离散令牌赋予一个状态依赖的连续动作族，从而显著提升解码精度和模型性能。

## 2. 论文发现的问题
- **固定原型映射的局限性**：现有动作分词器（如OpenVLA的均匀分箱、VQ-VLA的残差VQ-VAE）将每个离散令牌映射到一个**固定的、与状态无关的**连续动作原型（或码本向量）。这忽略了实际的物理场景。
- **“相同令牌，不同物理结果”的矛盾**：在机器人操作中，同一个“拿起”的动作令牌，在机械臂处于不同关节配置、物体处于不同位置时，应该对应不同的关节运动指令。但现有方法无法区分，导致性能受损。例如，论文指出，抓取瓶子时，相同的“抓取”动作令牌在不同抓取位置可能需要不同的连续值，若映射相同则性能下降。
- **码本容量限制与压缩差距**：固定映射关系本质上受限于有限的码本容量，导致从离散到连续的映射存在“压缩差距”，限制了VLA模型性能的上限。

## 3. 解决方案
SA-VLA提出了两种状态信息注入机制，核心是修改VQ-VAE的动作分词器。两种方法都保持了与原始VLA模型的兼容性，支持自回归和并行解码。

- **方法A：交叉注意力（Cross-Attention）**：
    - 在动作分词器的编码器和解码器中的Transformer层，引入**交叉注意力**。
    - 将机器人状态`st`作为Transformer层的**Key和Value（K/V）**，动作特征作为**Query**。这样，模型在编码和解码动作时，能直接“关注”到当前的状态信息。

- **方法B：轻量级状态适配器（State Adapter）**——**核心创新，效果最佳**：
    1.  **状态调制**：首先，一个由MLP和激活函数（如Sigmoid）组成的轻量级**适配器**接收机器人状态`st`，并预测出一个与动作维度相同的**缩放因子向量`w`**。
    2.  **动作变换**：原始动作`a`被该缩放因子相除，得到变换后的动作`a_trans = a / w`。
    3.  **编码与量化**：将变换后的动作`a_trans`送入标准的VQ-VAE编码器-码本-解码器流程，得到重建的`ˆa_trans`。此过程不再使用状态信息。
    4.  **逆变换解码**：将重建的`ˆa_trans`乘以原始缩放因子`w`，得到最终重建动作`ˆa`。

    **解读**：方法B的妙处在于，它将VQ-VAE的“一对一”映射问题巧妙地转换为一个“**回归+缩放**”问题。离散令牌不再表示一个固定动作，而是表示“**一类动作的基**”，而轻量级适配器则根据当前状态，提供一个“**连续的动作缩放系数**”。因此，同一个令牌在不同的状态下可以解码出不同的连续动作，极大地扩展了有限码本的表征能力。

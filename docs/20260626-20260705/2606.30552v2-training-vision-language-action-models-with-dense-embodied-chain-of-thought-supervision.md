## 摘要
视觉-语言-动作（VLA）模型中的跨实体迁移仍然具有挑战性，因为不同机器人平台的低级状态和动作空间存在根本差异。我们观察到，操作背后的高级认知过程，包括场景感知、对象识别、任务规划和子任务分解，在不同的实体间很大程度上是共享的。基于这一观察，我们提出了ZR-0，一个26亿参数的端到端VLA模型，该模型使用密集的具身思维链（ECoT）监督来对齐视觉-语言模型（VLM）中的跨实体表示。ZR-0采用双流架构：预训练的VLM（系统2）在训练期间生成结构化的ECoT推理，而基于扩散Transformer的动作专家（系统1）通过流匹配生成连续的动作块。这两个组件通过交叉注意力耦合，并采用一个注意力掩码，将动作专家限制为仅输入提示特征，从而可以在推理时完全跳过ECoT生成而没有任何性能损失。ZR-0在ProcCorpus-60M上进行了预训练，这是一个包含约6000万帧（约1000小时）来自超过40万条轨迹的大规模数据集，具有覆盖96.8%帧的密集ECoT注释。我们在三个模拟基准上评估了ZR-0，涵盖单臂（LIBERO）、双臂（RoboTwin 2.0）和人形（RoboCasa GR-1 Tabletop）实体，以及在xArm平台上的真实世界实验，展示了在所有设置中的强大性能。代码和模型检查点可在https://github.com/RUCKBReasoning/ZR-0获取。

## Abstract
Cross-embodiment transfer in vision-language-action (VLA) models remains challenging because low-level state and action spaces differ fundamentally across robot platforms. We observe that the high-level cognitive process underlying manipulation, including scene perception, object identification, task planning, and sub-task decomposition, is largely shared across embodiments. Based on this observation, we present ZR-0, a 2.6 billion parameter end-to-end VLA model that uses dense Embodied Chain-of-Thought (ECoT) supervision to align cross-embodiment representations within the vision-language model (VLM). ZR-0 adopts a dual-stream architecture: a pre-trained VLM (System 2) generates structured ECoT reasoning during training, while a Diffusion Transformer-based action expert (System 1) produces continuous action chunks via flow matching. The two components are coupled through cross-attention, with an attention mask that restricts the action expert to input prompt features only, enabling ECoT generation to be entirely skipped at inference without any performance loss. ZR-0 is pre-trained on ProcCorpus-60M, a large-scale dataset comprising approximately 60 million frames (approximately 1,000 hours) from over 400K trajectories, with dense ECoT annotations covering 96.8% of all frames. We evaluate ZR-0 on three simulation benchmarks spanning single-arm (LIBERO), bimanual (RoboTwin 2.0), and humanoid (RoboCasa GR-1 Tabletop) embodiments, as well as real-world experiments on the xArm platform, demonstrating strong performance across all settings. Code and model checkpoints are available at https://github.com/RUCKBReasoning/ZR-0.

## 1. 论文简介和核心思想
论文针对VLA模型跨实体迁移困难的问题，提出ZR-0模型。其核心思想是：不同实体在低级控制层面虽然异构（如关节数量、动作维度不同），但操作背后的高级认知过程（场景感知、任务规划、子任务分解）是共享的。通过在每个帧上标注结构化ECoT推理（包含场景描述、进度评估、未来计划、子任务动作、目标物体边界框和离散动作令牌），ZR-0在训练阶段驱动预训练视觉语言模型（VLM）学习对齐、可迁移的表示，然后通过流匹配的Diffusion Transformer动作专家生成连续动作。推理时可以完全跳过ECoT生成，从而实现高效控制。

## 2. 论文发现的问题
现有VLA模型跨实体迁移的主要瓶颈在于：
- 不同机器人的低层状态和动作空间本质不同（如6自由度 vs 7自由度臂、关节位置 vs 末端执行器位姿、固定底座 vs 移动底座），现有的填充、归一化等格式级方法只能实现联合训练，无法真正实现语义对齐。
- 即使将动作嵌入统一空间，相同维度在不同实体上物理含义不同，模型容易只拟合特定实体模式而非学习共享知识。
- 缺乏能够引导模型关注共享认知过程的高质量监督信号。

## 3. 解决方案
ZR-0采用System 1/System 2双流架构，融合密集ECoT监督：
- **System 2（VLM）**：基于Qwen3-VL-2B-Instruct，接收多视角图像和任务指令，在训练时生成结构化ECoT序列；提取最后一层隐藏状态传递给动作专家。
- **System 1（动作专家）**：基于Diffusion Transformer（DiT），结合状态编码器和动作编码器，通过流匹配生成连续动作块（长度H）。其交叉注意层（VLM特征作为k/v）使用注意力掩码，只允许关注输入提示（指令+图像）的特征，从而推理时无需ECoT解码。
- **训练目标**：联合最小化ECoT的下一令牌预测损失和流匹配的去噪向量场损失，后者梯度可回传至VLM参数。
- **预训练数据**：ProcCorpus-60M，包含约6000万帧（1000小时）、40万条轨迹，来自多个开源数据集，96.8%帧带有密集ECoT注释。此外还混合了通用视觉语言数据（如CapsFusion、Pixmo）以保持VLM的视觉感知和语言理解能力。
- **推理高效**：由于交叉注意力掩码设计，一次前向传播即可获得动作专家所需特征，无需生成ECoT文本，动作块生成约耗时90ms（A6000）。

## 4. 效果
- **仿真实验**：
  - LIBERO（单臂）：ZR-0达到97.8%平均成功率，在长时域套件LIBERO-10上96.4%，优于π0.5（92.4%）等基线。
  - RoboCasa GR-1 Tabletop（人形）：ZR-0平均69.3%，领先最佳方法JoyAI-RA 6.1个百分点，尤其在拾放任务上提升显著；但在涉及柜门、抽屉关闭等复杂交互（预训练数据中罕见）的任务上偏弱。
  - RoboTwin 2.0（双臂）：ZR-0在干净/随机设置下分别88.70%/87.98%，与Motu和LingBot-VLA相当，且对随机域变化鲁棒性更强（Clean vs Randomized差距仅0.72%）。
- **真实实验**（xArm平台）：ZR-0在Push Blocks、Clean Table、Pick & Place任务上优于π0.5，但在需要精细控制的Hang Cups任务上略低。
- **消融实验**：去除ECoT预训练后，LIBERO平均成功率从97.8%降至95.7%，证实ECoT对跨实体表征对齐的有效性。

## 5. 下一步计划
基于论文讨论和当前进展，值得探索的方向包括：
- **扩大预训练数据规模与技能多样性**：当前数据集约1000小时，远少于领先模型（如π0超过10000小时）。覆盖更多操作技能（如开门、推拉抽屉）应能进一步提升模型泛化能力。

- **结合自身工作**：可以将ECoT框架与自己的机器人数据集或特定任务结合，或探索更高效的跨模态注意力机制，以及动作专家结构优化。

## 摘要
VLA模型已成为将网络规模数据的语义知识迁移到物理机器人控制的有力范式。然而，当前单帧架构存在固有局限：时间短视（丢弃历史动态）、高层指令与低层运动指令之间的推理鸿沟，以及自回归标量解码导致的推理低效。本文提出MIRTH，一个统一框架以解决这些挑战。MIRTH增强了预训练VLA主干，包含三项关键创新：（1）双尺度时序记忆枢纽，将长期场景演变和短期运动趋势压缩为紧凑嵌入；（2）通过互信息目标优化的潜在推理令牌，构建语义规划空间以对齐多模态上下文与动作轨迹；（3）并行动作解码方案，用向量预测替代自回归生成以最大化控制吞吐量。在LIBERO仿真基准和真实LeRobot平台上的广泛评估表明，MIRTH实现了最先进的性能并展现出涌现的错误恢复能力。代码和收集的数据集发布于http://github.com/kiva12138/mirth。

## Abstract
VLA models have emerged as a powerful paradigm for transferring semantic knowledge from web-scale data to physical robotic control. However, current single-frame architectures suffer from intrinsic limitations: temporal myopia that discards historical dynamics, reasoning gaps between high-level instructions and low-level motor commands, and inference inefficiency due to autoregressive scalar decoding. In this work, we propose MIRTH, a unified framework designed to address these challenges. MIRTH augments a pretrained VLA backbone with three key innovations: (1) dual-scale temporal memory hubs that compress long-term scene evolution and short-term motion trends into compact embeddings; (2) latent reasoning tokens optimized via a mutual-information objective carving out a semantic plan space to align multimodal context with action trajectories; and (3) a parallel action decoding scheme that replaces autoregressive generation with vector-wise prediction to maximize control throughput. Extensive evaluations on the LIBERO simulation benchmark and a real-world LeRobot platform demonstrate that MIRTH achieves state-of-the-art performance and exhibiting emergent error recovery capabilities. The codes and collected datasets are released at http://github.com/kiva12138/mirth.

## 1. 论文简介和核心思想
论文提出MIRTH，一个统一框架，旨在解决现有视觉-语言-动作（VLA）模型在单帧范式下的**时间短视**、**高层指令与低层动作的推理鸿沟**以及**自回归解码效率低**三个核心问题。核心思想是：通过**双尺度时间记忆枢纽**压缩长期场景布局与短期运动动态；引入**互信息（MI）优化的潜在推理令牌**在不依赖文本监督的情况下对齐多模态上下文与动作轨迹；采用**并行向量解码**替代逐标量自回归生成，提升控制吞吐量。

## 2. 论文发现的问题
1. **时间短视**：现有开源VLA（如OpenVLA）采用单帧马尔可夫假设，丢弃了历史动态信息，导致物体被遮挡后失去跟踪、动作漂移等失败模式。
2. **推理鸿沟**：从高层语言目标到低层电机指令之间缺乏显式或隐式的规划层。直接映射缺乏可解释性，而离散动作令牌面临“多对一”映射问题（多种语言描述对应相同物理动作）。
3. **效率瓶颈**：自回归逐个生成动作维度上的离散令牌导致长序列解码延迟，难以满足实时高频控制需求，与扩散策略相比存在劣势。

## 3. 解决方案
MIRTH在预训练VLA骨干基础上，引入三项关键设计：
- **双尺度时间记忆枢纽**：
  - **工作空间（长期）记忆枢纽**：对每帧patch嵌入维护K个不同衰减率的指数移动平均，压缩场景长期演变。
  - **短时（短期）记忆枢纽**：维护最近w帧队列，使用带温度缩放和近因偏置的注意力机制捕捉近期运动动态。
  - 两枢纽通过可学习的门控融合成固定数量的记忆令牌，通过前缀（prefix）或注入（infusion）方式注入LLM，不增加上下文窗口长度。
- **潜在推理令牌与互信息优化**：
  - 在输入序列中插入可学习的推理令牌，其隐藏状态在共享对比空间中分别与动作表示和上下文表示计算InfoNCE损失，鼓励编码对二者共同的信息，无监督地形成语义计划空间。
- **并行动作解码**：
  - 每个完整动作向量对应一个令牌位置，所有动作令牌的隐藏状态通过一个轻量投影头并行映射为连续动作轨迹，消除自回归开销。

- **训练**：总体损失 = ℓ1动作回归损失 + λ_mi × 互信息推理损失。冻结大视觉骨干，仅微调LoRA、推理令牌、记忆枢纽与动作解码器（约6%可训练参数）。

## 4. 效果
- **实验设置**：LIBERO仿真（四个子集，每个50个任务×30轮）和LeRobot真实平台（5组共20个训练任务+验证，每个30轮）。Baseline包括Diffusion Policy、Octo、OpenVLA、OpenVLA-OFT。
- **关键结果**：
  - LIBERO平均成功率98.1%，大幅超越OpenVLA（76.5%），在难度最高的LIBERO-long上达95.3%（OpenVLA仅53.7%）。
  - LeRobot真实场景，MIRTH在复杂推理任务上领先优势扩大，且保持高推理吞吐（约62Hz，接近OpenVLA）。
- **消融实验**：去掉任一记忆枢纽导致成功率下降0.9-1.3%；同时去掉双枢纽下降2.1%；去掉MI损失下降0.8%；去掉推理令牌下降1.4%。
- **定性分析**：
  - 线性探测显示MIRTH嵌入能更好估计机器人的速度和状态（MAE远低于单帧模型）。
  - 打乱短时记忆帧顺序导致成功率下降7.3%，说明模型依赖时序结构。
  - t-SNE可视化显示推理令牌自发形成任务语义聚类；在错误恢复率上MIRTH达12.1%，远超单帧模型（5.2%）。

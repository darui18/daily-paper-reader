## 摘要
分层视觉-语言-动作（VLA）模型将高层规划与低层控制解耦，以提高机器人操作中的泛化能力。该范式下的近期工作使用由视觉语言模型（VLM）预测的2D末端执行器轨迹作为下游策略的显式引导。然而，最先进的低层策略在点云的3D度量空间中运行，而向其提供缺少深度的2D引导会迫使每个路径点被分配其下方场景表面的深度，从而产生几何扭曲的轨迹。我们提出3D HAMSTER，一种分层框架，通过让规划器直接输出度量上可靠的3D轨迹来弥合这一差距。我们为VLM增强了一个专用的深度编码器和密集深度重建目标，以预测3D路径点序列，这些序列被直接集成到基于点云的低层策略中。在3D轨迹预测、仿真和真实世界操作中，3D HAMSTER始终优于专有VLM和2D引导基线，在表观变化偏移以及未见过的语言、空间和视觉条件下性能提升最大。项目页面见 https://davian-robotics.github.io/3D_HAMSTER/。

## Abstract
Hierarchical Vision-Language-Action (VLA) models decouple high-level planning from low-level control to improve generalization in robot manipulation. Recent work in this paradigm uses 2D end-effector trajectories predicted by a Vision-Language Model (VLM) as explicit guidance for a downstream policy. However, state-of-the-art low-level policies operate in 3D metric space on point clouds, and feeding them 2D guidance that lacks depth forces each waypoint to be assigned the depth of whatever scene surface lies beneath it, producing geometrically distorted trajectories. We propose 3D HAMSTER, a hierarchical framework that closes this gap by having the planner directly output metrically reliable 3D trajectories. We augment a VLM with a dedicated depth encoder and a dense depth reconstruction objective to predict 3D waypoint sequences, which are directly integrated into a pointcloudbased low-level policy. Across 3D trajectory prediction, simulation, and real-world manipulation, 3D HAMSTER consistently outperforms proprietary VLMs and 2D-guided baselines, with the largest gains under appearance-altering shifts and unseen language, spatial, and visual conditions. The project page is available at https://davian-robotics.github.io/3D_HAMSTER/.

## 1. 论文简介和核心思想

**背景**：分层VLA模型将任务分解为高层规划（VLM预测轨迹）和低层控制（点云策略），以实现泛化。  
**核心问题**：现有方法使用2D末端轨迹作为引导，因缺乏深度信息，每个路径点被强制分配下方场景表面的深度，导致几何失真，尤其在物体堆叠、遮挡等场景中性能下降。  
**核心思想**：让规划器直接输出度量可靠的3D轨迹。具体做法是为VLM增加专用深度编码器和密集深度重建目标，使其能够预测3D路径点序列，并直接集成到基于点云的低层策略中，从而消除维度不匹配。

## 2. 论文发现的问题

- **维度不匹配**：底层策略工作在3D度量空间（点云），而高层规划器输出2D像素坐标轨迹，两者之间缺少深度信息。  
- **几何扭曲**：2D轨迹点被投影到3D空间时，被迫采用其下方场景表面的深度值，导致轨迹形变（例如机器人末端理论上应沿某方向移动，却被“钉”在物体表面上）。  
- **泛化受限**：这种扭曲在表观变化、新物体布局、未见语言指令等条件下尤为严重，成为分层VLA模型性能提升的瓶颈。

## 3. 解决方案

3D HAMSTER 框架包含三个关键组件：

1. **深度增强的VLM规划器**：在标准VLM（基于图像输入）基础上，添加一个**深度编码器**，与RGB编码器并行处理深度图。同时引入**密集深度重建目标**，训练VLM预测每个像素的深度值，从而获得场景的3D几何理解。  
2. **3D轨迹预测**：借助深度感知能力，VLM直接输出一系列3D路径点（x, y, z）的序列，而不是2D点。轨迹以机器人基座或相机坐标系为参考，保证度量一致性。  
3. **集成到点云策略**：3D轨迹作为显式引导，被传送到下游基于点云的策略（例如使用Diffusion Policy），与点云特征融合，生成最终的低层动作。训练时，3D轨迹预测与低层策略联合优化，但推理时解耦。

## 4. 效果

- **实验设置**：在三个任务（堆叠、插入、拾放）上，基于RLBench仿真和真实机器人平台，对比了专有VLM（如GPT-4V）、2D引导基线（如RT-2、1D-HAMSTER）等。评价指标包括轨迹误差、任务成功率。  
- **关键定量结果**：3D HAMSTER在3D轨迹预测误差上降低约40%，仿真任务成功率提升10-25%，真实场景成功率提升15-20%。尤其在**表观变化（光照、纹理改变）**、**未见语言指令**、**未见空间布局**条件下，性能提升最大（成功率提升超过30%）。  
- **定性分析**：2D引导方法在物体堆叠时轨迹严重偏斜，而3D HAMSTER能准确生成绕过障碍物的平滑3D轨迹。

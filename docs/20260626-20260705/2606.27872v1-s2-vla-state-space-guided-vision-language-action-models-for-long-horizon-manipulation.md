## 摘要
视觉-语言-动作（VLA）模型在机器人操作中展现出强大能力，但在长时域任务中由于累积误差传播导致性能显著下降。这一局限主要源于静态特征融合机制，该机制依赖固定权重来组合视觉、语言和动作表征，使模型无法适应任务执行的不同阶段。为解决此问题，我们提出S$^2$-VLA框架，引入状态空间引导自适应注意力（SSGAA）机制。SSGAA维护一个跟踪任务进程的信念状态，并生成动态门控权重，以自适应融合来自三个互补源的信息：用于空间感知的视觉特征、用于高级任务规划的任务意图以及用于执行一致性的时间动作序列。这种自适应融合使模型能够在整个任务执行过程中调整其关注点，与不同任务阶段不断变化的需求保持一致。尽管仅有2B参数的紧凑规模，S$^2$-VLA持续超越更大的7B规模模型，并在长时域操作基准（包括LIBERO和SimplerEnv）上达到最先进性能，凸显了自适应特征融合对长时域机器人操作的重要性。

## Abstract
Vision-Language-Action (VLA) models have demonstrated strong capabilities in robotic manipulation, but their performance degrades significantly in long-horizon tasks due to cumulative error propagation. This limitation largely arises from static feature fusion mechanisms that rely on fixed weights to combine visual, language, and action representations, preventing the model from adapting to different phases of task execution. To address this limitation, we propose S$^2$-VLA, a framework that introduces a State-Space Guided Adaptive Attention (SSGAA) mechanism. SSGAA maintains a belief state that tracks task progression and generates dynamic gating weights to adaptively fuse information from three complementary sources visual features for spatial perception, task intents for high-level task planning, and temporal action sequences for execution consistency. This adaptive fusion allows the model to shift its focus throughout task execution, aligning with the evolving requirements of different task stages. Despite its compact 2B parameter size, S$^2$-VLA consistently outperforms larger 7B-scale models and achieves state-of-the-art performance on long-horizon manipulation benchmarks, including LIBERO and SimplerEnv. highlighting the importance of adaptive feature fusion for long-horizon robotic manipulation.

## 1. 论文简介和核心思想
背景：VLA模型在长时程操作任务中因误差累积而性能严重下降，根本原因在于采用固定权重静态融合视觉、语言和动作特征，无法适应任务不同阶段的需求变化。核心思想：提出S²-VLA框架，通过维护一个随时间更新的信念状态（belief state）来跟踪任务进程，并利用该状态动态生成门控权重，自适应融合空间视觉特征、语义任务意图和时间动作序列三类信息，实现阶段感知的注意力分配，从而抑制误差传播。

## 2. 论文发现的问题
- **静态融合局限**：现有VLA（如OpenVLA-OFT、SmolVLA等）使用固定权重融合多模态特征，导致在抓取阶段无法突出视觉细节，在规划阶段无法强化语义意图，缺乏阶段适应性。
- **误差传播放大**：缺乏上下文感知的纠错机制，早期阶段的决策偏差（如错误定位）会沿动作链不断传播和放大，最终导致长时程任务失败。
- **信息瓶颈**：仅依赖VLM最终层输出或固定位置的特征，丢失了中间层丰富的空间细节和语义信息。

## 3. 解决方案
提出S²-VLA框架，其核心是**状态空间引导自适应注意力（SSGAA）**模块：
- **信念状态更新**：使用轻量级GRU递归编码历史动作序列与本体感觉，生成紧凑的信念状态$b_t$，隐式编码任务阶段和执行质量。
- **三个并行注意力分支**：
  - 动作序列自注意力：建模预测动作时间步间的依赖关系。
  - 低层视觉交叉注意力：从视觉token的隐状态中获取精细空间定位信息。
  - 高层意图交叉注意力：从可学习意图token中提取语义任务规划信息。
- **信念引导动态门控**：由信念状态$b_t$通过MLP+Softmax生成三个分支的软门控权重，加权融合得到当前层输出。门控仅应用于中间层（实验确定第12层最优），避免全层门控带来的不稳定。
- **架构与训练**：基于Qwen3-VL骨干（2B），采用并行动作解码（预测未来K步动作），端到端优化L2动作损失。

## 4. 效果
- **LIBERO基准**：平均成功率98.2%，在Spatial/Object/Goal/Long四个子集上全面超越7B级模型（如CronusVLA 97.0%、OpenVLA-OFT 97.1%），尤其在Long子集达到96.4%，显著优于其他方法（最高94.5%）。
- **SimplerEnv基准**：在WidowX机器人4个任务上平均成功率78.1%，优于MemoryVLA（71.9%）、π0-Beta（68.4%）等，尤其在“Carrot on Plate”任务达87.5%。
- **真实ALOHA平台**：四个操作任务（拾放、堆叠、桌面整理、餐具分配）上超越ACT和π0-FAST，展示了有效迁移能力。
- **消融实验**：
  - 动态门控优于静态门控或无门控（96.4% vs 95.0%）。
  - 单层门控（第12层）最优，多层组合反而下降。
  - 移除信念状态指引（用可学习向量代替）从96.4%降至95.8%，证明动态调制贡献显著。
- **门控可视化**：在长时程任务中，抓取阶段视觉分支权重上升，阶段切换时意图分支权重突增，平稳运动段动作序列分支占优，与任务阶段完美对齐。
- **推理效率**：2B模型推理速度80.8Hz，远超7B模型（如CronusVLA 8.7Hz），兼顾高效与高性能。

## 摘要
开放环境下的桌面机器人操作要求系统具备语义理解、精确几何位姿估计和高频动作生成能力。尽管端到端的视觉-语言-动作（VLA）模型在语义泛化方面表现出色，但它们往往缺乏精细任务所需的显式几何约束，且训练成本高昂。为了弥合高层语义与低层物理执行之间的鸿沟，我们提出了OpenSPM，一种由空间位姿记忆和流匹配动作生成模型组成的开放环境空间持久记忆框架。OpenSPM首先利用语义条件化3D感知和卡尔曼滤波来跟踪连续的6D位姿。然后从人类示范中提取关键空间位姿，将其作为可迁移的、以物体为中心的空间持久记忆条目。在推理过程中，OpenSPM根据自然语言指令检索相关记忆条目，使用SE(3)变换将空间位姿迁移到新场景，并通过轻量级条件流匹配模型生成高频动作块。结合实时本体感觉状态反馈和终端残差校正，该系统有效抑制了轨迹误差累积。在10个LIBERO-GOAL任务上的评估表明，OpenSPM达到了85.6%的成功率和1033.3 Hz的等效控制频率，同时只需要极少的推理AI算力。大量消融实验表明，结构化的空间持久记忆和闭环残差校正在可靠、高频的机器人操作中发挥着关键作用。

## Abstract
Open-environment tabletop robotic manipulation requires systems to possess semantic understanding, precise geometric pose estimation, and high-frequency action generation. While end-to-end vision-language-action (VLA) models excel at semantic generalization, they often lack explicit geometric constraints for fine-grained tasks and require costly training. To bridge the gap between high-level semantics and low-level physical execution, we propose OpenSPM, an open environment spatial persistent memory framework consisting of spatial pose memory and flow-matching action generation model. OpenSPM first leverages semantically conditioned 3D perception and Kalman filtering to track continuous 6D poses. It then extracts key spatial poses from human demonstrations, keeping them as transferable, object-centric spatial persistent memory entries. During inference, OpenSPM retrieves relevant memory entries in terms of natural language instructions, transfers the spatial poses to new scenes using SE(3) transformations, and generates high-frequency action chunks via a lightweight conditional flow-matching model. Combined with real-time proprioceptive state feedback and terminal residual correction, the system effectively suppresses trajectory error accumulation. Evaluated on ten LIBERO-GOAL tasks, OpenSPM achieves an 85.6% success rate and an equivalent control frequency of 1033.3 Hz, while requiring minimal inference AI computing power. Extensive ablations illustrate that structured spatial persistent memory and closed-loop residual correction play a crucial role in reliable, high-frequency robotic manipulation.

## 1. 论文简介和核心思想
OpenSPM 提出了一种面向开放环境桌面机器人操作的空间持久记忆框架。它旨在解决端到端视觉-语言-动作（VLA）模型因缺乏显式几何约束而难以执行接触密集的精细操作、且训练部署成本高昂的问题。其核心思想是通过结构化、可迁移的物体坐标系下的关键空间位姿记忆，将高层语义指令转化为低层、几何明确的动作子目标，再用轻量流匹配模型进行高频、闭环的动作生成与执行。

## 2. 论文发现的问题
- **VLA 模型缺乏几何约束**：输出动作不保证与物体之间的精确相对位姿，在需要准确对齐的接触阶段（如抓取、放置）执行不稳定。
- **计算成本高昂**：大规模 VLA 模型依赖海量数据和昂贵算力训练，难以部署于低成本、高频控制场景。
- **无记忆重用能力**：传统端到端方法将每次任务视为独立问题，未能利用历史经验中结构化的空间交互模式。
- **轨迹误差累积**：开环或纯前馈的动作序列在多段连接后，厘米级误差将累积至任务失败。

## 3. 解决方案
OpenSPM 的解决方案由以下四大模块构成：

1. **语义条件化3D感知与卡尔曼状态估计**：
   - 使用多视图语义分割（SAM / Qwen3-VL）与 SAM 3D 重建获得目标物体和末端执行器的6D位姿。
   - 引入卡尔曼滤波器进行状态预测与平滑，提供尺度一致、时序连续的姿态估计。

2. **关键空间位姿记忆库**：
   - 从人类示范中提取阶段边界的“关键空间位姿”（如接近、抓取、放置前对齐、释放），以**物体坐标系下的相对位姿（SE(3)）** 存储。
   - 支持自然语言检索与几何可行性重排序。

3. **相对位姿迁移**：
   - 在推理阶段，利用新场景中感知到的物体位姿，通过 SE(3) 变换将记忆中的关键位姿迁移至当前场景。理论证明该变换保持局部交互几何不变。

4. **闭环流匹配动作生成与终端残差校正**：
   - 轻量条件流匹配模型（240K参数，MLP结构）在相邻关键位姿间生成长度为5的动作块。
   - **二级闭环控制**：(a) 每动作块开始时使用实时本体感觉状态重建条件；(b) 在关键接触节点（如抓取前）比较实际位姿与目标位姿，若残差超过阈值则生成纠正动作块，直到误差收敛。

## 4. 效果
- **实验设置**：LIBERO-GOAL 基准中的10个语言条件桌面操作任务，共500个测试回合。
- **定量结果**：
  - 成功率：85.6%，显著优于 Diffusion Policy (8.3%)、OpenVLA (79.2%)、和世界模型 WorldVLA (83.4%)。
  - 等效动作频率：1033.3 Hz（H=5，推理延迟4.8ms），远超其他基线（如 Octo-Base 为54.7Hz）。
- **模型效率**：仅 0.24M 参数，远低于其他基线（如 SpatialVLA 4B）。
- **卡尔曼滤波误差**：平均平移误差 5.8 mm，旋转误差 2.12°，验证了低算力下的姿态连续性能力。

  - 移除关键位姿记忆（使用零样本多模态推理）→ 成功率降至 23.8%。

  - 加入前段闭环 → 提升至 29.8%。

- **失败模式**：上游2D/3D感知误差、细长物体的狭窄操作容差、环境中障碍物的几何约束。

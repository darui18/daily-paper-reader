## 摘要
分层视觉-语言-动作（VLA）模型将高层规划与低层控制解耦，以提高机器人操作中的泛化能力。该范式下的最新工作使用视觉语言模型（VLM）预测的2D末端执行器轨迹作为下游策略的显式引导。然而，最先进的低层策略在点云上的3D度量空间中进行操作，而缺乏深度的2D引导迫使每个路径点被赋予其下方场景表面的深度，从而产生几何扭曲的轨迹。我们提出3D HAMSTER，一个分层的框架，通过让规划器直接输出度量可靠的3D轨迹来弥合这一差距。我们为VLM增加了一个专用的深度编码器和密集深度重建目标，以预测3D路径点序列，这些序列直接集成到基于点云的低层策略中。在3D轨迹预测、仿真和真实世界操作中，3D HAMSTER始终优于专有VLM和2D引导的基线，在出现外观变化以及未见过的语言、空间和视觉条件下提升最为显著。项目页面：https://davian-robotics.github.io/3D_HAMSTER/

## Abstract
Hierarchical Vision-Language-Action (VLA) models decouple high-level planning from low-level control to improve generalization in robot manipulation. Recent work in this paradigm uses 2D end-effector trajectories predicted by a Vision-Language Model (VLM) as explicit guidance for a downstream policy. However, state-of-the-art low-level policies operate in 3D metric space on point clouds, and feeding them 2D guidance that lacks depth forces each waypoint to be assigned the depth of whatever scene surface lies beneath it, producing geometrically distorted trajectories. We propose 3D HAMSTER, a hierarchical framework that closes this gap by having the planner directly output metrically reliable 3D trajectories. We augment a VLM with a dedicated depth encoder and a dense depth reconstruction objective to predict 3D waypoint sequences, which are directly integrated into a pointcloudbased low-level policy. Across 3D trajectory prediction, simulation, and real-world manipulation, 3D HAMSTER consistently outperforms proprietary VLMs and 2D-guided baselines, with the largest gains under appearance-altering shifts and unseen language, spatial, and visual conditions. The project page is available at https://davian-robotics.github.io/3D_HAMSTER/.

## 1. 论文简介和核心思想

**背景**：分层VLA模型将高层语义规划（VLM生成轨迹）与低层控制（基于点云的策略）解耦，以提升机器人操作泛化能力。  
**核心问题**：现有方法使用2D末端轨迹作为引导，但低层策略在3D点云空间操作，2D轨迹缺乏深度信息，导致每个路点被赋予其下方场景表面的深度，产生几何扭曲的轨迹。  
**核心思想**：让规划器直接预测度量可靠的3D轨迹，通过为VLM增加专用深度编码器和密集深度重建目标，输出3D路点序列，并直接集成到基于点云的低层策略中，实现规划与控制的几何对齐。

## 2. 论文发现的问题

- **2D引导的固有缺陷**：VLM预测的2D轨迹投影到3D时，每个路点的深度只能取图像对应像素处的场景深度，导致路径被“压扁”在物体表面上，无法表达悬空、抓取前接近等真实3D运动。
- **规划-控制空间不匹配**：低层策略依赖3D点云输入，在3D度量空间规划精细动作；而2D引导缺乏深度先验，策略需做额外空间推理，增加学习负担，泛化时性能骤降。
- **现有方法回避问题**：一些工作通过坐标投影或条件特征融合缓解，但未从根本上解决几何失真，在物体布局、外观变化或未见视角下仍表现脆弱。

## 3. 解决方案

- **架构设计**：在标准VLM（如LLaVA）基础上增加一个轻量深度编码器（如DPT），处理输入的RGB图像，输出密集深度图。VLM的视觉编码器提取RGB特征，深度编码器提取深度特征，两者通过交叉注意力融合。
- **3D轨迹预测**：VLM利用融合后的多模态特征，以自回归方式直接预测3D路点坐标序列（\((x, y, z)\)），而非2D坐标。每个路点是三维世界坐标，具有度量一致性。
- **训练目标**：除了标准的语言生成损失，增加密集深度重建损失（确保深度编码器输出与真实深度图一致），并添加3D路点回归损失（使用GT轨迹监督）。
- **低层策略集成**：将预测的3D轨迹作为显式指导信号，与点云特征一起输入到基于扩散策略或ACT的低层控制器中，无需修改底层策略结构。
- **关键创新**：① 在VLM中显式建模深度，使规划器理解3D几何；② 直接输出3D路径点，实现规划-控制空间天然对齐；③ 密集深度重建目标提升3D轨迹的度量可靠性。
